REPL in python is (read evaluate print loop) used for run python 
In cmd like >>>

>>> # Use the REPL as a simple calculator
>>> 5 + 10
15
>>> 2 * 2 / 8
0.5
>>> 
>>> # Assign variables and use them
>>> greeting = "Hello, REPL!"
>>> print(greeting)
Hello, REPL!
